Presentations listed here have been presented at past CSG meetings, primarily by CSG officers and members. These presentations are provided for academic usage only.

Binary Exploitation

Binary Exploitation is the manipulation of binary executables to produce possibly malicious behavior. Standard binary exploit topics include:
  • Buffer Overflows
  • Format String Vulnerabilities
  • Fuzzing
  • Shellcode
  • Return Oriented Programming

Cryptography

Cryptography is the study of secure communication techniques. Standard cryptography topics include:
  • Classical Cryptography
  • Symmetric Cryptography
  • Asymmetric Cryptography
  • Hashing

Digital Forensics

Digital Forensics is the pracitce of uncovering deleted and hidden data. Standard digital forensics topics include:
  • Data Recovery
  • Metadata Analysis
  • Server Fingerprinting
  • Traffic Inspection

Malware Analysis

Malware Anaylsis is the study of malicious code. Standard malware analysis topics include:
  • Malware Creation
  • Malware Analysis Countermeasures
  • Packing
  • Obfuscation

Network Security

Network Security is the practice of securing communications between multiple computers. Standard network security topics include:
  • Intrusion Prevention Systems
  • Intrusion Detection Systems
  • Firewalls
  • Network Pivoting

Reverse Engineering

Reversing Engineer is the practice of introspecting binary executables. Standard reverse engineer topics include:
  • Static Analysis
  • Dynamic Anaylsis

System Security

System security is the hardening of a system against would-be attackers. Standard system security topics include:
  • System Hardening
  • Intrusion Detection
  • Intrusion Prevention

Web Exploitation

Web Exploitation is a class of malicious attacks on web servers. Standard web exploitation topics include:
  • SQL Injection
  • Cross Site Scripting
  • Cross Site Request Forgery
  • Local File Includes
